# 髙橋勇喜 Portfolio Site

就活用ポートフォリオサイトの静的HTML版です。

## 構成

- `index.html`：サイト本体
- `style.css`：デザイン
- `script.js`：スクロールアニメーション、スマホメニュー、GitHub API取得

## 公開前に変更する箇所

1. `index.html` の `your-email@example.com` を実際のメールアドレスへ変更してください。
2. `index.html` の OGP URL `https://takahashi_pf.jp/` を実際の公開URLへ変更してください。
3. GitHub API欄は、サイト上でユーザー名を入力すると公開リポジトリを取得できます。
4. `ogp.png` を追加する場合は、リポジトリ直下に画像を配置してください。

## GitHub Pagesでの公開手順

1. GitHubで新規リポジトリを作成します。
2. このフォルダ内のファイルをアップロードします。
3. Settings → Pages → Branch を `main` / root に設定します。
4. 表示されたURLでアクセス確認します。
